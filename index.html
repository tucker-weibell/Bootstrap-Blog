Bootstrap For Dummies

If you're completely new to Web Development and you're looking to get your feet wet in easy and customizable styles, then you've come to the right place. We will be covering the basics of Bootstrap including how to download, customize, and use Bootstrap in all your projects. This is not a comprehensive guide covering all the different Bootstrap components and templates. Rather, this is a simple guide to help dummies like you and me, get started with using Bootstrap in web applications.

Why use Bootstrap?

Bootstrap was first released in 2011, and since then Bootstrap has become very popular in web development. Why is bootrap so popular? Because Web designers and web developers find it extremely flexible and easy to work with. Some big advantages to Bootstrap are that it is responsive, meaning, it's compatible with wide browsers as well as smaller mobile browsers, it allows for consistent design with re-usable components, and it is not difficult to learn how to use. These benifits allow web developers to focus on the actual development of their applications rather than the design.

How does it work?

Bootstrap is basically just a large CSS file. If you are unfamiliar with CSS I've included some links at the end of this article that will explain HTML5, CSS, and JavaScript so go check those out first.

The idea is that instead of you needing to style your entire page with your own CSS file, you can just use Bootstrap. Bootstrap can be included in your html by adding a link to the external Bootstrap in the <head> section of your html document or by downloading the Bootstrap source code files and adding them to your project. In this demo we will primarily focus on importing the external Bootstrap in the <head> of our documents.



Getting Started

Getting started with Bootstrap is super easy!

Let's begin by opening up a text editor (I prefer VS Code but you can use whatever text editor suites you best) and we will create a basic skeleton of an HTML page. Title the page "index.html".

<!DOCTYPE html>

<html>

      <head>

      </head>

      <body>

      </body>

</html>


Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load bootstrap CSS. Go to Bootstrap Introduction for latest bootstrap link. 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-BmbxuPwQa2lc/FVzBcNJ7UAyJxM6wuqIj61tLrc4wSX0szH/Ev+nYRRuWlolflfl" crossorigin="anonymous">


Many of the Bootstrap components require Javascript to function. Paste the <script> inside the <body> tags at the very bottom. Go to Bootstrap Introduction for latest javascript link. 

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta2/dist/js/bootstrap.bundle.min.js" integrity="sha384-b5kHyXgcpbZJO/tY9Ul7kGkf1S0CWuKcCD38l8YkeH8z8QjE0GmW1gYU5S9FOnJ0" crossorigin="anonymous"></script>


Your index.html should now look like this: 

<!DOCTYPE html>
    <html>

        <head>
            <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-BmbxuPwQa2lc/FVzBcNJ7UAyJxM6wuqIj61tLrc4wSX0szH/Ev+nYRRuWlolflfl" crossorigin="anonymous">
        </head>

        <body>

            <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta2/dist/js/bootstrap.bundle.min.js" integrity="sha384-b5kHyXgcpbZJO/tY9Ul7kGkf1S0CWuKcCD38l8YkeH8z8QjE0GmW1gYU5S9FOnJ0" crossorigin="anonymous"></script>
        </body>

    </html>


Great! All done! You are now ready to begin using Bootstrap templates and elements in your code. In the next section we will go over how to use bootstrap navbars and forms. Navbars and Forms are two special Bootstrap templates (css code) that will make it easy for you to contruct pre-styled navbars and forms on your own website. There are hundreds of different Bootstrap templates so when you are done with this quick tutorial head over to the Bootstrap Website and start learning how to included other fun templates in your html code. Be creative!



Navbars

By definition, "A navigation bar is a user interface element within a webpage that contains links to other sections of the website. In most cases, the navigation bar is part of the main website template, which means it is displayed on most, if not all, pages within the website. This means that no matter what page you are viewing, you can use the navigation bar to visit other sections of the website."


If you wanted to, you could create your own html navbar and use CSS to style it, but but Bootstrap makes navbars really easy by giving you several different templates to choose from that are already styled for you. In the navbar tutorial we will be making a simple navbar like the one below using bootstrap.


Navbar

In about 1 more minute you will have a fully functional navbar. Simply head to the Bootstrap Navbar Documentation and copy the template for the any one of the navbars you like. Below is an example of one of the navbar templates. You may copy this if you wish. 

<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
    <a class="navbar-brand" href="#">Navbar</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">
        <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarSupportedContent">
        <ul class="navbar-nav me-auto mb-2 mb-lg-0">
        <li class="nav-item">
            <a class="nav-link active" aria-current="page" href="#">Home</a>
        </li>
        <li class="nav-item">
            <a class="nav-link" href="#">Link</a>
        </li>
        <li class="nav-item dropdown">
            <a class="nav-link dropdown-toggle" href="#" id="navbarDropdown" role="button" data-bs-toggle="dropdown" aria-expanded="false">
            Dropdown
            </a>
            <ul class="dropdown-menu" aria-labelledby="navbarDropdown">
            <li><a class="dropdown-item" href="#">Action</a></li>
            <li><a class="dropdown-item" href="#">Another action</a></li>
            <li><hr class="dropdown-divider"></li>
            <li><a class="dropdown-item" href="#">Something else here</a></li>
            </ul>
        </li>
        <li class="nav-item">
            <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Disabled</a>
        </li>
        </ul>
        <form class="d-flex">
        <input class="form-control me-2" type="search" placeholder="Search" aria-label="Search">
        <button class="btn btn-outline-success" type="submit">Search</button>
        </form>
    </div>
    </div>
</nav>


Once you have selected a template to use for your navbar, simply paste the navbar html code between the body tags of your index.html document. Vwa-Lah!! You can now open your index.html in your browser and you will see a navbar at the top of the page.



Forms

By definition, "a webform, web form or HTML form on a web page allows a user to enter data that is sent to a server for processing. Forms can resemble paper or database forms because web users fill out the forms using checkboxes, radio buttons, or text fields. For example, forms can be used to enter shipping or credit card data to order a product, or can be used to retrieve search results from a search engine."


Creating forms in html that look neat and function like they are supposed to can be difficult. Designing a webapp should be fun and easy! Thanks to Bootstrap we can create a simple pre-styled form in under 5 minutes. Are you ready? Let's start!

We are not done yet though. As you can see your form looks a little strange and spans the entire width of the page. That's not what we want so we are going to do a little bit of custom styling of our own to clean this form up a bit.


To customize our Bootstrap template we are going to create a CSS file of our own. We will be pasting the link to our CSS style sheet below the Bootstrap stylesheet link. It is important that your CSS stylesheet link is pasted below the Bootstrap link so that your style modifications override the Boostrap default styles.

Begin by navigating to the same folder as your index.html. Once inside the folder create a document called style.css. To link your CSS to your index.html page, paste this code in the <head> section of your code BELOW the Bootstrap link. 

<link href="style.css" rel="stylesheet">                                               


In your index.html find the opening <form> tag and inside of the tag add class="myform". Your opening form tag should look like this.

<form class="myform row g-3">                                                                         

Now open the style.css and add the following code. 

.myform {
    margin-top: 40px;
    margin-left: 20%;
    margin-right: 20%;
    border: 0.5px solid gray;
    padding: 10px;
}

Make sure to save both the index.html and the style.css and then open index.html in your browser and refresh the page. You will see that your stylesheet now overrides the css from Bootstrap and allowed you to customize the form by changing the form width, padding, and margins, and adding a thin border around the form.
